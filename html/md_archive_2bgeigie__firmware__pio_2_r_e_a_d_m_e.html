bGeigie-Zen firmware (ESP32)

Quick Start

Pre-requisites: python3

  1. Install Platform IO CLI 
     python3 -c "$(curl -fsSL https://raw.githubusercontent.com/platformio/platformio/master/scripts/get-platformio.py)"
    I haven't tried this step myself, but if it fails, try to follow the instructions on platform.io's website. Rob is using platform.io within VS Code. The link for that is here
  2. Clone the repository 
     git clone git@github.com:Safecast/bGeigiezen.git
  3. Change branch to Robin dev 
     cd bGeigiezen
 git checkout robin/M5Stack_bGeigieRaku_V2.0
  4. Change directory 
     cd bgeigie_firmware_pio
  5. Connect the device using USB, compile, and upload 
     pio run -t upload
